Chapter 2

Probability

2.1 — Sample Space

Experiment: Any process that generates a set of data.

Sample Space S: The set of ALL possible outcomes of a statistical experiment.

Sample Point: Each individual outcome in the sample space.

  • Discrete S — finite or countably infinite outcomes (e.g., coin toss: {H, T})
  • Continuous S — an interval of real numbers (e.g., recycle time: S = {x | 1.5 < x < 5})
Example 2.1 Sample Space — Coin + Die

Flip a coin; if HEAD flip again; if TAIL toss a die.

S = {HH, HT, T1, T2, T3, T4, T5, T6}

Example 2.2 Sample Space — 3 Messages

Messages classified as on-time (o) or late (l). Classify the next 3 messages.

S = {ooo, ool, olo, oll, loo, lol, llo, lll} — 2³ = 8 outcomes

2.2 — Events

An event is any subset of the sample space S.

Union (OR)A ∪ B = all outcomes in A OR B (or both)
Intersection (AND)A ∩ B = all outcomes in BOTH A AND B
Complement (NOT)A' = all outcomes in S that are NOT in A

Mutually Exclusive: A ∩ B = ∅ → events cannot occur simultaneously.

Event Laws Commutative: A ∩ B = B ∩ A, A ∪ B = B ∪ A
Distributive: (A ∪ B) ∩ C = (A ∩ C) ∪ (B ∩ C)
Associative: (A ∪ B) ∪ C = A ∪ (B ∪ C)
DeMorgan's: (A ∪ B)' = A' ∩ B' and (A ∩ B)' = A' ∪ B'
Complement: (A')' = A

2.3 — Counting Sample Points

① Multiplication Rule

Total ways for k steps n₁ × n₂ × n₃ × ··· × nₖ
Example Computer Assembly

Sam chooses: 2 chips × 4 hard drives × 3 memory × 5 accessories.

2 × 4 × 3 × 5 = 120 ways
Example License Plates (letters repeated)

3 letters followed by 3 digits. (a) Repetition allowed. (b) Not allowed.

(a) 26³ × 10³ = 17,576,000

(b) 26 × 25 × 24 × 10 × 9 × 8 = 11,232,000

② Permutations (Order Matters)

All n distinct objectsn! = n × (n-1) × (n-2) × ··· × 1
n objects taken r at a timeₙPᵣ = n! / (n − r)!
n objects with repeats (n₁,n₂,...,nₖ)n! / (n₁! × n₂! × ··· × nₖ!)

Use permutations when ORDER MATTERS — passwords, rankings, scheduling, arrangements.

Example License Plate — Hit & Run

Plates: RLS4 _ _ _ (last 3 digits different, first digit fixed as 4; remaining pool: 0–9 except 4 = 9 digits)

₉P₃ = 9! / 6! = 9 × 8 × 7 = 504
Example ENGINEERING (indistinguishable letters)

11 letters: E×3, N×3, G×2, I×2, R×1. Total distinct arrangements?

S = 11! / (3! × 3! × 2! × 2! × 1!) = 39,916,800 / 144 = 277,200

P(spelling ENGINEERING) = 1 / 277,200 ≈ 0.00036%

③ Combinations (Order Does NOT Matter)

n objects choose r ₙCᵣ = C(n,r) = n! / (r! × (n − r)!)

Use combinations when ORDER DOESN'T MATTER — committees, selections, groups, hands of cards.

Example Club Committee

A club has 9 members. How many committees of 3 can be formed?

C(9,3) = 9! / (3! × 6!) = 84
Example Defective Parts

From a bin of 50 parts (3 defective, 47 non-defective), 6 are selected randomly. P(exactly 2 defective)?

Total ways = C(50,6) = 15,890,700

Favorable = C(3,2) × C(47,4) = 3 × 178,365 = 535,095

2.4 — Probability of an Event

Axioms0 ≤ P(A) ≤ 1 | P(∅) = 0 | P(S) = 1
Equally Likely OutcomesP(A) = (# outcomes in A) / (total # outcomes N)

2.5 — Additive Rules

General (2 events) P(A ∪ B) = P(A) + P(B) − P(A ∩ B)

Mutually Exclusive P(A ∪ B) = P(A) + P(B) [since P(A ∩ B) = 0]

Three events (Inclusion-Exclusion) P(A ∪ B ∪ C) = P(A) + P(B) + P(C) − P(A∩B) − P(A∩C) − P(B∩C) + P(A∩B∩C)

Complement Rule P(A') = 1 − P(A)

Example Combined Probability — P(A)=0.5, P(B)=0.4, P(A'∩B')=0.2

Find: (a) P(A∪B), (b) P(A∩B), (c) P(A'∩B), (d) P(A∩B')

(a) P(A∪B) = 1 − P(A'∩B') = 1 − 0.2 = 0.8

(b) P(A∩B) = P(A) + P(B) − P(A∪B) = 0.5 + 0.4 − 0.8 = 0.1

(c) P(A'∩B) = P(B) − P(A∩B) = 0.4 − 0.1 = 0.3

(d) P(A∩B') = P(A) − P(A∩B) = 0.5 − 0.1 = 0.4

2.6 — Conditional Probability, Independence & Product Rule

Conditional Probability P(B | A) = P(A ∩ B) / P(A) [P(A) > 0]

Product Rule (General) P(A ∩ B) = P(A) × P(B | A) = P(B) × P(A | B)

Independence — ALL three equivalent P(A | B) = P(A) OR P(B | A) = P(B) OR P(A ∩ B) = P(A)·P(B)

Multiple Independent Events P(E₁ ∩ E₂ ∩ ··· ∩ Eₖ) = P(E₁) × P(E₂) × ··· × P(Eₖ)

Series vs. Parallel Systems

Series System

ALL components must work. Fails if ANY one fails.

P(system works) = P(A) × P(B) × ···

Reliability always decreases with more components.

Parallel System

Fails only if ALL components fail. Works if ANY one works.

P(system fails) = P(A fails) × P(B fails) × ···

Reliability always increases with more components.

Random Variables & Probability Distributions

3.1 — Concept of a Random Variable

A random variable is a function that assigns a real number to each outcome in the sample space.

  • Discrete RV — finite or countably infinite range; obtained by counting (e.g., # defectives, # heads)
  • Continuous RV — an interval of real numbers for its range; obtained by measuring (e.g., voltage, weight, time)

3.2 — Discrete Probability Distributions

Probability Mass Function (PMF) — f(x) = P(X = x) (1) f(x) ≥ 0 for all x
(2) Σ f(x) = 1
(3) P(X = x) = f(x)
Cumulative Distribution Function (CDF) F(x) = P(X ≤ x) = Σ_{xᵢ ≤ x} f(xᵢ)

Properties: 0 ≤ F(x) ≤ 1 | Non-decreasing
P(a < X ≤ b) = F(b) − F(a)

3.3 — Continuous Probability Distributions

Probability Density Function (PDF) — f(x) (1) f(x) ≥ 0 for all x
(2) ∫₋∞^∞ f(x) dx = 1
(3) P(a < X < b) = ∫ₐ^b f(x) dx

Note P(X = any single point) = 0 [for continuous RV]
CDF for Continuous RV F(x) = P(X ≤ x) = ∫₋∞^x f(t) dt
f(x) = dF(x)/dx

Mathematical Expectation

4.1 — Mean (Expected Value)

Expected Value of X Discrete: μ = E(X) = Σ x · f(x)
Continuous: μ = E(X) = ∫₋∞^∞ x · f(x) dx

Expected Value of a Function g(X) Discrete: E[g(X)] = Σ g(x) · f(x)
Continuous: E[g(X)] = ∫₋∞^∞ g(x) · f(x) dx

Two Variables g(X,Y) Discrete: E[g(X,Y)] = ΣΣ g(x,y) · f(x,y)
Continuous: E[g(X,Y)] = ∬ g(x,y) · f(x,y) dx dy

4.2 — Variance & Covariance

Variance σ² = E[(X − μ)²] = E(X²) − μ²

Discrete: σ² = Σ (x − μ)² f(x)
Continuous: σ² = ∫₋∞^∞ (x − μ)² f(x) dx

Shortcut Formula σ² = E(X²) − [E(X)]² ← use this one on exams!

Standard Deviation σ = √σ²

4.3 — Linear Combinations of Random Variables

Mean Rules E(aX + b) = aE(X) + b
E(X ± Y) = E(X) ± E(Y)
E(XY) = E(X)·E(Y) [only if X and Y are INDEPENDENT]

Variance Rules Var(aX + b) = a²·Var(X) [constant b doesn't affect spread]
Var(aX + bY) = a²Var(X) + b²Var(Y) [only if X,Y INDEPENDENT]
σ²(aX+b) = a²σ²

Covariance σ_XY = E[(X−μX)(Y−μY)] = E(XY) − μX·μY
If X and Y are independent: σ_XY = 0

5.3 — Hypergeometric Distribution

Setup N = total population
k = number of successes in population
n = sample size (without replacement)
x = number of successes in sample

PMF h(x; N, n, k) = C(k,x) · C(N−k, n−x) / C(N,n)
where max(0, n−N+k) ≤ x ≤ min(k, n)

Mean & Variance μ = nk/N
σ² = n·(k/N)·(1−k/N)·(N−n)/(N−1)

Key difference from Binomial: Hypergeometric is sampling WITHOUT replacement from a finite population. When N is large (n/N ≤ 0.05), approximate with Binomial using p = k/N.

6.1 — Continuous Uniform Distribution

PDF f(x; A, B) = 1/(B−A) for A ≤ x ≤ B, 0 elsewhere

Mean & Variance μ = (A+B)/2
σ² = (B−A)²/12

CDF F(x) = 0 for x < A
F(x) = (x−A)/(B−A) for A ≤ x ≤ B
F(x) = 1 for x > B

Probability in interval [a,b] ⊆ [A,B] P(a < X < b) = (b−a)/(B−A)
Example 6.20 Coffee Machine — A=7, B=10

X ~ Uniform(7,10). Find (a) μ, σ² (b) P(X < 8.5)

(a) μ = (7+10)/2 = 8.5, σ² = (10−7)²/12 = 9/12 = 0.75

(b) P(X < 8.5) = (8.5−7)/(10−7) = 1.5/3 = 0.5

6.2–6.4 — Normal Distribution

PDF n(x; μ, σ) = (1/√(2πσ²)) · exp[−(x−μ)²/(2σ²)] for −∞ < x < ∞

Properties • Bell-shaped, symmetric about μ
• Mode = Mean = Median = μ
• Total area under curve = 1
• Approaches x-axis asymptotically

Standardization — Convert X to Z Z = (X − μ) / σ where Z ~ N(0,1)

Finding Probabilities P(X < x) = P(Z < z) — look up in Z-table
P(a < X < b) = P(z₁ < Z < z₂) = Φ(z₂) − Φ(z₁)
P(X > x) = 1 − Φ(z)

Z-table gives P(Z < z) = Φ(z). For negative z: Φ(−z) = 1 − Φ(z) by symmetry.

Example 6.5 X ~ N(300, 50). P(X > 362)?

z = (362 − 300)/50 = 1.24

P(X > 362) = 1 − Φ(1.24) = 1 − 0.8925

= 0.1075

6.6 — Exponential Distribution

Definition Time/distance between successive Poisson events with rate λ.

PDF f(x) = λ·e^(−λx) for x ≥ 0

CDF F(x) = P(X ≤ x) = 1 − e^(−λx)
P(X > x) = e^(−λx)

Mean & Variance μ = E(X) = 1/λ
σ² = 1/λ² ← NOTE: σ = 1/λ = μ (mean = std. deviation)

Lack of Memory Property P(X < t₁+t₂ | X > t₁) = P(X < t₂)
"The exponential distribution has no memory of the past."
Example Checkout Counter — λ=2 customers/min

1. Find mean and variance of waiting time.
2. P(at least 1 customer waiting when 3-min service completes)?
3. P(customer arrives within 1 min)?
4. P(arrives next minute | no one arrived in last 2 min)?

1. μ = 1/2 = 0.5 min, σ² = 1/4 = 0.25

2. P(y ≤ 3) = 1 − e^(−2×3) = 1 − e^(−6) = 0.9975

3. P(y ≤ 1) = 1 − e^(−2×1) = 1 − e^(−2) = 0.8647

4. By lack-of-memory property: same as above = 0.8647

Example Computer Log-ons — λ=25/hour

What is the probability of no log-ons in the next 6 minutes (0.1 hour)?

P(X > 0.1) = e^(−25 × 0.1) = e^(−2.5)

= 0.0821
Example Log-ons — P(between 2 and 3 minutes)

λ=25/hr. P(0.033 hr < X < 0.05 hr)?

= F(0.05) − F(0.033) = (1−e^(−25×0.05)) − (1−e^(−25×0.033))

= e^(−0.833) − e^(−1.25)

= 0.435 − 0.287 = 0.148
Example Geiger Counter — E(X)=1.4 min. P(particle in 30 sec)?

λ = 1/1.4. P(X ≤ 0.5)?

P(X ≤ 0.5) = 1 − e^(−0.5/1.4) = 1 − e^(−0.357)

= 1 − 0.700 = 0.300

Q2Additive Rule · Union1.5 marks
In a class of 80 students, 46 use the bus, 32 use carpooling, and 14 use both. What is the probability that a randomly selected student uses the bus or carpooling?
a) 0.625
b) 0.700
c) 0.750
d) 0.800 ✓
e) 0.925

Let B = bus, C = carpooling. Additive rule:

P(B ∪ C) = P(B) + P(C) − P(B ∩ C) = 46/80 + 32/80 − 14/80 = 64/80

= 0.800
Q3Complement Rule · Independent Events1.5 marks
Each manufactured component is defective with probability 0.10, independently of the others. What is the probability that at least one of 3 components is defective?
a) 0.100
b) 0.271 ✓
c) 0.300
d) 0.729
e) 0.900

"At least one defective" = complement of "none defective".

P(none defective) = (0.90)³ = 0.729

P(at least 1 defective) = 1 − 0.729

= 0.271

Trap: d) 0.729 is P(none defective) — the complement, not the answer.

Q4Conditional Probability1.5 marks
Of 60 trainees, 24 attended a workshop and 12 of those who attended received an excellent rating. If one workshop attendee is selected at random, what is the probability that the trainee received an excellent rating?
a) 0.20
b) 0.30
c) 0.40
d) 0.50 ✓
e) 0.60

We select from workshop attendees only (24 people). Of those, 12 got excellent ratings.

P(Excellent | Attended) = 12 / 24

= 0.50

The total of 60 trainees is irrelevant — we are told one attendee is selected, so denominator = 24.

Q5Independence · P(A ∩ B̄)1.5 marks
Events A and B are independent with P(A) = 0.50 and P(B) = 0.40. Find P(A ∩ B̄).
a) 0.10
b) 0.20
c) 0.30 ✓
d) 0.40
e) 0.50

Since A and B are independent, A and B̄ are also independent.

P(A ∩ B̄) = P(A) × P(B̄) = P(A) × (1 − P(B)) = 0.50 × 0.60

= 0.30

Alternative: P(A ∩ B̄) = P(A) − P(A ∩ B) = 0.50 − (0.50 × 0.40) = 0.50 − 0.20 = 0.30

Q6Bayes' Theorem1.5 marks
A warehouse receives 60% of its parts from Supplier A and 40% from Supplier B. The defective rates are 2% and 5%, respectively. If a randomly selected part is defective, what is the probability that it came from Supplier A?
a) 0.250
b) 0.375 ✓
c) 0.500
d) 0.625
e) 0.750

P(A) = 0.60, P(B) = 0.40. P(D|A) = 0.02, P(D|B) = 0.05.

Step 1 — Total P(D): P(D) = P(D|A)·P(A) + P(D|B)·P(B)

= (0.02)(0.60) + (0.05)(0.40) = 0.012 + 0.020 = 0.032

Step 2 — Bayes: P(A|D) = P(D|A)·P(A) / P(D) = 0.012 / 0.032

= 0.375

A random variable X has the distribution: f(x) = k (x=0), 2k (x=1), 3k (x=2), 4k (x=3). What is k?

The random variable X has values 0, 1, and 2 with probabilities 0.20, 0.50, and 0.30, respectively. Find E[X].
a) 0.8
b) 1.0
c) 1.1 ✓
d) 1.3

Q9Variance — Shortcut Formula1.5 marks
A random variable X has values −1, 0, and 1 with probabilities 0.25, 0.50, and 0.25, respectively. Find Var(X).
a) 0.25
b) 0.50 ✓
c) 0.75
d) 1.00
e) 1.50

E[X] = (−1)(0.25) + 0(0.50) + 1(0.25) = −0.25 + 0 + 0.25 = 0

E[X²] = (1)(0.25) + 0(0.50) + (1)(0.25) = 0.25 + 0 + 0.25 = 0.50

Var(X) = E[X²] − (E[X])² = 0.50 − 0²

= 0.50

This distribution is symmetric about 0, so E[X]=0. Variance measures spread — the values ±1 each occur with probability 0.25, giving Var = 0.50.

Q10Linear Transformation — E[aX + b]1.5 marks
If E[X] = 3 and Y = 5X + 2, find E[Y].
a) 10
b) 15
c) 17 ✓
d) 18
e) 25

Linearity rule: E[aX + b] = a · E[X] + b

E[Y] = 5 · E[X] + 2 = 5(3) + 2 = 15 + 2

= 17
Q11Standard Deviation of Linear Transformation1.5 marks
If the standard deviation of X is 2 and Y = 4X − 1, what is the standard deviation of Y?
a) 2
b) 4
c) 7
d) 8 ✓
e) 15

Var(aX + b) = a² · Var(X) — the constant b has NO effect on spread.

Var(Y) = 4² · Var(X) = 16 × (2²) = 16 × 4 = 64

SD(Y) = √64

= 8

Shortcut: SD(aX + b) = |a| · SD(X) = |4| × 2 = 8. The −1 shifts the distribution but does not stretch it.

Q12CDF · Finding P(X = x) from F(x)1.5 marks
For a discrete random variable X with CDF: F(x) = 0 (x<1), 0.25 (1≤x<2), 0.65 (2≤x<3), 1 (x≥3). Find P(X = 2).
a) 0.20
b) 0.25
c) 0.40 ✓
d) 0.65
e) 0.90

For discrete RVs: P(X = x) = F(x) − F(x−1)

P(X = 2) = F(2) − F(1) = 0.65 − 0.25

= 0.40

F(2) = P(X ≤ 2) = 0.65. F(1) = P(X ≤ 1) = 0.25. The difference strips out P(X ≤ 1), leaving only P(X = 2).

Q13Identifying the Correct Distribution1.5 marks
An assembly line records the number of scratched items among 20 independently produced items, where each item has the same probability of being scratched. Which distribution models the number of scratched items?
a) Binomial ✓
b) Poisson
c) Geometric
d) Hypergeometric
e) Exponential

Check all 4 Binomial conditions:

① Fixed n = 20 trials ✓

② Binary outcomes: scratched or not scratched ✓

③ Constant p (same probability for each item) ✓

④ Independent trials ✓

→ Binomial Distribution

Hypergeometric would apply if sampling without replacement from a finite population. Poisson would apply for random events per unit time/area (no fixed n).

Q14Binomial · P(X = x)1.5 marks
A quality inspector checks 5 products independently, where each product is defective with probability 0.20. What is the probability of finding exactly one defective product?
a) 0.2048
b) 0.3277
c) 0.4096 ✓
d) 0.5904
e) 0.6723

X ~ Binomial(n = 5, p = 0.20). PMF: P(X = x) = C(n,x) · pˣ · (1−p)^(n−x)

P(X = 1) = C(5,1) · (0.20)¹ · (0.80)⁴ = 5 × 0.20 × 0.4096

= 0.4096
Q15Binomial · Complement (At Least One)1.5 marks
A printer fails on any job with probability 0.10, independently. What is the probability of at least one failure in 4 print jobs?
a) 0.2439
b) 0.3439 ✓
c) 0.4096
d) 0.6561
e) 0.9000

X ~ Binomial(n = 4, p = 0.10). Use the complement trick:

P(X ≥ 1) = 1 − P(X = 0) = 1 − (0.90)⁴ = 1 − 0.6561

= 0.3439

Trap: d) 0.6561 = P(no failures) — the complementary value. Never confuse P(X=0) with P(X≥1).

Q16Poisson · P(X = 0)1.5 marks
The number of calls arriving at a help desk in one minute follows a Poisson distribution with mean 2. What is the probability of receiving no calls in a minute?
a) 0.0498
b) 0.1353 ✓
c) 0.2707
d) 0.4060
e) 0.8647

X ~ Poisson(λ = 2). PMF: P(X = x) = e⁻λ · λˣ / x!

P(X = 0) = e⁻² · 2⁰ / 0! = e⁻²

= 0.1353

Key values: e⁻¹ ≈ 0.3679 · e⁻² ≈ 0.1353 · e⁻³ ≈ 0.0498 · e⁻⁵ ≈ 0.0067. Note: a) 0.0498 = e⁻³ (wrong λ).

Q17Poisson · P(X ≤ 1)1.5 marks
The number of faults on a sheet follows a Poisson distribution with mean 3. What is the probability that a sheet has at most one fault?
a) 0.0498
b) 0.1494
c) 0.1991 ✓
d) 0.4232
e) 0.8009

X ~ Poisson(λ = 3). "At most one" = P(X ≤ 1) = P(X = 0) + P(X = 1).

P(X = 0) = e⁻³ = 0.0498

P(X = 1) = e⁻³ · 3¹ / 1! = 3e⁻³ = 3 × 0.0498 = 0.1494

P(X ≤ 1) = 0.0498 + 0.1494

= 0.1991

Shortcut: P(X ≤ 1) = e⁻λ(1 + λ) = e⁻³(1 + 3) = 4 × 0.0498 = 0.1991

Q18Geometric · First Success on Trial x1.5 marks
A machine produces an acceptable item with probability 0.25 on each independent trial. What is the probability that the first acceptable item occurs on the 3rd trial?
a) 0.0469
b) 0.1055
c) 0.1406 ✓
d) 0.1875
e) 0.4219

X ~ Geometric(p = 0.25). "First success on trial x": f(x) = (1−p)^(x−1) · p

P(X = 3) = (1 − 0.25)^(3−1) · 0.25 = (0.75)² × 0.25 = 0.5625 × 0.25

= 0.1406

Interpretation: trials 1 and 2 fail (probability 0.75 each), trial 3 succeeds (0.25).

Q19Negative Binomial · k-th Success1.5 marks
A salesperson makes a sale with probability 0.50 on each independent call. What is the probability that the 2nd sale occurs on the 4th call?
a) 0.0625
b) 0.1250
c) 0.1875 ✓
d) 0.2500
e) 0.3750

X ~ Negative Binomial(k = 2, p = 0.50). PMF: f(x) = C(x−1, k−1) · pᵏ · (1−p)^(x−k)

P(X = 4) = C(4−1, 2−1) · (0.50)² · (0.50)^(4−2)

= C(3,1) · (0.50)⁴ = 3 × 0.0625

= 0.1875

Interpretation: among the first 3 calls, exactly 1 was a sale (C(3,1) arrangements), then call 4 is definitely a sale.

Q20Hypergeometric · Expected Value1.5 marks
A box contains 40 batteries, of which 10 are defective. If 8 batteries are selected without replacement, what is the expected number of defective batteries selected?
a) 1
b) 2 ✓
c) 2.5
d) 4
e) 8

Sampling without replacement from a finite population → Hypergeometric.

N = 40 (total), k = 10 (defective in population), n = 8 (sample size).

E(X) = nk/N = (8 × 10) / 40 = 80/40

= 2

Intuition: 10/40 = 25% of the population is defective, so we expect 25% of 8 = 2 to be defective.

a) 0.20

Q22Continuous Uniform · Inverse CDF1.5 marks
A delivery time X is uniformly distributed from 0 to 20 minutes. Find the time t such that P(X ≤ t) = 0.75.
a) 5 minutes
b) 10 minutes
c) 12 minutes
d) 15 minutes ✓
e) 18 minutes

X ~ Uniform(A = 0, B = 20). CDF: F(t) = P(X ≤ t) = (t − A)/(B − A) = t/20

Set F(t) = 0.75:   t/20 = 0.75  →  t = 0.75 × 20

= 15 minutes

General formula for inverse CDF of Uniform: t = A + p·(B − A), where P(X ≤ t) = p.

Q23Exponential · P(X > x) given mean1.5 marks
The lifetime of a sensor follows an exponential distribution with mean 5 years. What is the probability that it lasts more than 10 years?
a) 0.1353 ✓
b) 0.2707
c) 0.3679
d) 0.6321
e) 0.8647

Mean = 1/λ = 5 years  →  λ = 1/5 = 0.2 per year.

P(X > x) = e^(−λx)

P(X > 10) = e^(−0.2 × 10) = e^(−2) = e^(−10/5)

= 0.1353

Shortcut: when asked for P(X > k·μ), the answer is e^(−k). Here k = 10/5 = 2, so e^(−2) = 0.1353.

Q24Exponential · P(X < x) given rate λ1.5 marks
The time until the next customer arrives is exponentially distributed with rate 0.5 per minute. Find P(X < 4).
a) 0.1353
b) 0.3679
c) 0.6321
d) 0.8647 ✓
e) 0.9817

X ~ Exponential(λ = 0.5). CDF: P(X < x) = 1 − e^(−λx)

P(X < 4) = 1 − e^(−0.5 × 4) = 1 − e^(−2) = 1 − 0.1353

Q25Normal · P(X < x) — One-Tailed1.5 marks
Test scores are normally distributed with mean 60 and standard deviation 10. What is P(X < 70)?
a) 0.1587
b) 0.3413
c) 0.5000
d) 0.6826
e) 0.8413 ✓

Standardize: Z = (X − μ)/σ = (70 − 60)/10 = 1.0

P(X < 70) = P(Z < 1.0) = Φ(1.0)

= 0.8413

70 is exactly 1 standard deviation above the mean. Key Φ values: Φ(1)=0.8413, Φ(2)=0.9772, Φ(−1)=0.1587.

Q26Normal · Symmetric Interval (μ ± 1σ)1.5 marks
Package weights are normally distributed with mean 100 g and standard deviation 15 g. What is P(85 < X < 115)?
a) 0.1587
b) 0.3174
c) 0.5000
d) 0.6826 ✓
e) 0.8413

Standardize both bounds:

z₁ = (85 − 100)/15 = −1.0     z₂ = (115 − 100)/15 = +1.0

P(85 < X < 115) = Φ(1.0) − Φ(−1.0) = 0.8413 − 0.1587

= 0.6826

Classic rule: P(μ − σ < X < μ + σ) = 0.6826 for any normal. Note: 85 = 100 − 15 and 115 = 100 + 15, so this is exactly ±1σ.

Q27Continuous PDF · Finding Constant c1.5 marks
A continuous random variable X has probability density function f(x) = cx for 0 < x < 2, and 0 otherwise. Find c.
a) 0.25
b) 0.50 ✓
c) 1.00
d) 1.50
e) 2.00

For a valid PDF, the total area must equal 1: ∫₋∞^∞ f(x) dx = 1

∫₀² cx dx = 1  →  c · [x²/2]₀² = 1  →  c · (4/2) = 1  →  2c = 1

c = 0.50

Strategy: always integrate f(x) over its full domain, set equal to 1, solve for the unknown constant. Same idea as Q7 for discrete PMFs.

Q28Continuous PDF · P(X > x) by Integration1.5 marks
A continuous random variable X has density f(x) = 2x for 0 < x < 1, and 0 otherwise. Find P(X > 0.5).
a) 0.25
b) 0.50
c) 0.625
d) 0.75 ✓
e) 0.875

P(X > 0.5) = ∫₀.₅¹ 2x dx = [x²]₀.₅¹ = 1² − (0.5)² = 1 − 0.25

= 0.75
