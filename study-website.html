Chapter 2

Probability

2.1 — Sample Space

Experiment: Any process that generates a set of data.

Sample Space S: The set of ALL possible outcomes of a statistical experiment.

Sample Point: Each individual outcome in the sample space.

  • Discrete S — finite or countably infinite outcomes (e.g., coin toss: {H, T})
  • Continuous S — an interval of real numbers (e.g., recycle time: S = {x | 1.5 < x < 5})
Example 2.1 Sample Space — Coin + Die

Flip a coin; if HEAD flip again; if TAIL toss a die.

S = {HH, HT, T1, T2, T3, T4, T5, T6}

Example 2.2 Sample Space — 3 Messages

Messages classified as on-time (o) or late (l). Classify the next 3 messages.

S = {ooo, ool, olo, oll, loo, lol, llo, lll} — 2³ = 8 outcomes

2.2 — Events

An event is any subset of the sample space S.

Union (OR)A ∪ B = all outcomes in A OR B (or both)
Intersection (AND)A ∩ B = all outcomes in BOTH A AND B
Complement (NOT)A' = all outcomes in S that are NOT in A

Mutually Exclusive: A ∩ B = ∅ → events cannot occur simultaneously.

Event Laws Commutative: A ∩ B = B ∩ A, A ∪ B = B ∪ A
Distributive: (A ∪ B) ∩ C = (A ∩ C) ∪ (B ∩ C)
Associative: (A ∪ B) ∪ C = A ∪ (B ∪ C)
DeMorgan's: (A ∪ B)' = A' ∩ B' and (A ∩ B)' = A' ∪ B'
Complement: (A')' = A

2.3 — Counting Sample Points

① Multiplication Rule

Total ways for k steps n₁ × n₂ × n₃ × ··· × nₖ
Example Computer Assembly

Sam chooses: 2 chips × 4 hard drives × 3 memory × 5 accessories.

2 × 4 × 3 × 5 = 120 ways
Example License Plates (letters repeated)

3 letters followed by 3 digits. (a) Repetition allowed. (b) Not allowed.

(a) 26³ × 10³ = 17,576,000

(b) 26 × 25 × 24 × 10 × 9 × 8 = 11,232,000

② Permutations (Order Matters)

All n distinct objectsn! = n × (n-1) × (n-2) × ··· × 1
n objects taken r at a timeₙPᵣ = n! / (n − r)!
n objects with repeats (n₁,n₂,...,nₖ)n! / (n₁! × n₂! × ··· × nₖ!)

Use permutations when ORDER MATTERS — passwords, rankings, scheduling, arrangements.

Example License Plate — Hit & Run

Plates: RLS4 _ _ _ (last 3 digits different, first digit fixed as 4; remaining pool: 0–9 except 4 = 9 digits)

₉P₃ = 9! / 6! = 9 × 8 × 7 = 504
Example ENGINEERING (indistinguishable letters)

11 letters: E×3, N×3, G×2, I×2, R×1. Total distinct arrangements?

S = 11! / (3! × 3! × 2! × 2! × 1!) = 39,916,800 / 144 = 277,200

P(spelling ENGINEERING) = 1 / 277,200 ≈ 0.00036%

③ Combinations (Order Does NOT Matter)

n objects choose r ₙCᵣ = C(n,r) = n! / (r! × (n − r)!)

Use combinations when ORDER DOESN'T MATTER — committees, selections, groups, hands of cards.

Example Club Committee

A club has 9 members. How many committees of 3 can be formed?

C(9,3) = 9! / (3! × 6!) = 84
Example Defective Parts

From a bin of 50 parts (3 defective, 47 non-defective), 6 are selected randomly. P(exactly 2 defective)?

Total ways = C(50,6) = 15,890,700

Favorable = C(3,2) × C(47,4) = 3 × 178,365 = 535,095

2.4 — Probability of an Event

Axioms0 ≤ P(A) ≤ 1 | P(∅) = 0 | P(S) = 1
Equally Likely OutcomesP(A) = (# outcomes in A) / (total # outcomes N)

2.5 — Additive Rules

General (2 events) P(A ∪ B) = P(A) + P(B) − P(A ∩ B)

Mutually Exclusive P(A ∪ B) = P(A) + P(B) [since P(A ∩ B) = 0]

Three events (Inclusion-Exclusion) P(A ∪ B ∪ C) = P(A) + P(B) + P(C) − P(A∩B) − P(A∩C) − P(B∩C) + P(A∩B∩C)

Complement Rule P(A') = 1 − P(A)

Example Combined Probability — P(A)=0.5, P(B)=0.4, P(A'∩B')=0.2

Find: (a) P(A∪B), (b) P(A∩B), (c) P(A'∩B), (d) P(A∩B')

(a) P(A∪B) = 1 − P(A'∩B') = 1 − 0.2 = 0.8

(b) P(A∩B) = P(A) + P(B) − P(A∪B) = 0.5 + 0.4 − 0.8 = 0.1

(c) P(A'∩B) = P(B) − P(A∩B) = 0.4 − 0.1 = 0.3

(d) P(A∩B') = P(A) − P(A∩B) = 0.5 − 0.1 = 0.4

2.6 — Conditional Probability, Independence & Product Rule

Conditional Probability P(B | A) = P(A ∩ B) / P(A) [P(A) > 0]

Product Rule (General) P(A ∩ B) = P(A) × P(B | A) = P(B) × P(A | B)

Independence — ALL three equivalent P(A | B) = P(A) OR P(B | A) = P(B) OR P(A ∩ B) = P(A)·P(B)

Multiple Independent Events P(E₁ ∩ E₂ ∩ ··· ∩ Eₖ) = P(E₁) × P(E₂) × ··· × P(Eₖ)

Series vs. Parallel Systems

Series System

ALL components must work. Fails if ANY one fails.

P(system works) = P(A) × P(B) × ···

Reliability always decreases with more components.

Parallel System

Fails only if ALL components fail. Works if ANY one works.

P(system fails) = P(A fails) × P(B fails) × ···

Reliability always increases with more components.

Bayes' Theorem P(Aᵢ|B) = P(B|Aᵢ)·P(Aᵢ) / [Σⱼ P(B|Aⱼ)·P(Aⱼ)]

Random Variables & Probability Distributions

3.1 — Concept of a Random Variable

A random variable is a function that assigns a real number to each outcome in the sample space.

  • Discrete RV — finite or countably infinite range; obtained by counting (e.g., # defectives, # heads)
  • Continuous RV — an interval of real numbers for its range; obtained by measuring (e.g., voltage, weight, time)

3.2 — Discrete Probability Distributions

Probability Mass Function (PMF) — f(x) = P(X = x) (1) f(x) ≥ 0 for all x
(2) Σ f(x) = 1
(3) P(X = x) = f(x)
Cumulative Distribution Function (CDF) F(x) = P(X ≤ x) = Σ_{xᵢ ≤ x} f(xᵢ)

Properties: 0 ≤ F(x) ≤ 1 | Non-decreasing
P(a < X ≤ b) = F(b) − F(a)

3.3 — Continuous Probability Distributions

Probability Density Function (PDF) — f(x) (1) f(x) ≥ 0 for all x
(2) ∫₋∞^∞ f(x) dx = 1
(3) P(a < X < b) = ∫ₐ^b f(x) dx

Note P(X = any single point) = 0 [for continuous RV]
CDF for Continuous RV F(x) = P(X ≤ x) = ∫₋∞^x f(t) dt
f(x) = dF(x)/dx

Mathematical Expectation

4.1 — Mean (Expected Value)

Expected Value of X Discrete: μ = E(X) = Σ x · f(x)
Continuous: μ = E(X) = ∫₋∞^∞ x · f(x) dx

Expected Value of a Function g(X) Discrete: E[g(X)] = Σ g(x) · f(x)
Continuous: E[g(X)] = ∫₋∞^∞ g(x) · f(x) dx

Two Variables g(X,Y) Discrete: E[g(X,Y)] = ΣΣ g(x,y) · f(x,y)
Continuous: E[g(X,Y)] = ∬ g(x,y) · f(x,y) dx dy

4.2 — Variance & Covariance

Variance σ² = E[(X − μ)²] = E(X²) − μ²

Discrete: σ² = Σ (x − μ)² f(x)
Continuous: σ² = ∫₋∞^∞ (x − μ)² f(x) dx

Shortcut Formula σ² = E(X²) − [E(X)]² ← use this one on exams!

Standard Deviation σ = √σ²

4.3 — Linear Combinations of Random Variables

Mean Rules E(aX + b) = aE(X) + b
E(X ± Y) = E(X) ± E(Y)
E(XY) = E(X)·E(Y) [only if X and Y are INDEPENDENT]

Variance Rules Var(aX + b) = a²·Var(X) [constant b doesn't affect spread]
Var(aX + bY) = a²Var(X) + b²Var(Y) [only if X,Y INDEPENDENT]
σ²(aX+b) = a²σ²

Covariance σ_XY = E[(X−μX)(Y−μY)] = E(XY) − μX·μY
If X and Y are independent: σ_XY = 0

5.3 — Hypergeometric Distribution

Setup N = total population
k = number of successes in population
n = sample size (without replacement)
x = number of successes in sample

PMF h(x; N, n, k) = C(k,x) · C(N−k, n−x) / C(N,n)
where max(0, n−N+k) ≤ x ≤ min(k, n)

Mean & Variance μ = nk/N
σ² = n·(k/N)·(1−k/N)·(N−n)/(N−1)

Key difference from Binomial: Hypergeometric is sampling WITHOUT replacement from a finite population. When N is large (n/N ≤ 0.05), approximate with Binomial using p = k/N.

6.1 — Continuous Uniform Distribution

PDF f(x; A, B) = 1/(B−A) for A ≤ x ≤ B, 0 elsewhere

Mean & Variance μ = (A+B)/2
σ² = (B−A)²/12

CDF F(x) = 0 for x < A
F(x) = (x−A)/(B−A) for A ≤ x ≤ B
F(x) = 1 for x > B

Probability in interval [a,b] ⊆ [A,B] P(a < X < b) = (b−a)/(B−A)
Example 6.20 Coffee Machine — A=7, B=10

X ~ Uniform(7,10). Find (a) μ, σ² (b) P(X < 8.5)

(a) μ = (7+10)/2 = 8.5, σ² = (10−7)²/12 = 9/12 = 0.75

(b) P(X < 8.5) = (8.5−7)/(10−7) = 1.5/3 = 0.5

6.2–6.4 — Normal Distribution

PDF n(x; μ, σ) = (1/√(2πσ²)) · exp[−(x−μ)²/(2σ²)] for −∞ < x < ∞

Properties • Bell-shaped, symmetric about μ
• Mode = Mean = Median = μ
• Total area under curve = 1
• Approaches x-axis asymptotically

Standardization — Convert X to Z Z = (X − μ) / σ where Z ~ N(0,1)

Finding Probabilities P(X < x) = P(Z < z) — look up in Z-table
P(a < X < b) = P(z₁ < Z < z₂) = Φ(z₂) − Φ(z₁)
P(X > x) = 1 − Φ(z)

Z-table gives P(Z < z) = Φ(z). For negative z: Φ(−z) = 1 − Φ(z) by symmetry.

Example 6.5 X ~ N(300, 50). P(X > 362)?

z = (362 − 300)/50 = 1.24

P(X > 362) = 1 − Φ(1.24) = 1 − 0.8925

= 0.1075

6.6 — Exponential Distribution

Definition Time/distance between successive Poisson events with rate λ.

PDF f(x) = λ·e^(−λx) for x ≥ 0

CDF F(x) = P(X ≤ x) = 1 − e^(−λx)
P(X > x) = e^(−λx)

Mean & Variance μ = E(X) = 1/λ
σ² = 1/λ² ← NOTE: σ = 1/λ = μ (mean = std. deviation)

Lack of Memory Property P(X < t₁+t₂ | X > t₁) = P(X < t₂)
"The exponential distribution has no memory of the past."
Example Checkout Counter — λ=2 customers/min

1. Find mean and variance of waiting time.
2. P(at least 1 customer waiting when 3-min service completes)?
3. P(customer arrives within 1 min)?
4. P(arrives next minute | no one arrived in last 2 min)?

1. μ = 1/2 = 0.5 min, σ² = 1/4 = 0.25

2. P(y ≤ 3) = 1 − e^(−2×3) = 1 − e^(−6) = 0.9975

3. P(y ≤ 1) = 1 − e^(−2×1) = 1 − e^(−2) = 0.8647

4. By lack-of-memory property: same as above = 0.8647

Example Computer Log-ons — λ=25/hour

What is the probability of no log-ons in the next 6 minutes (0.1 hour)?

P(X > 0.1) = e^(−25 × 0.1) = e^(−2.5)

= 0.0821
Example Log-ons — P(between 2 and 3 minutes)

λ=25/hr. P(0.033 hr < X < 0.05 hr)?

= F(0.05) − F(0.033) = (1−e^(−25×0.05)) − (1−e^(−25×0.033))

= e^(−0.833) − e^(−1.25)

= 0.435 − 0.287 = 0.148
Example Geiger Counter — E(X)=1.4 min. P(particle in 30 sec)?

λ = 1/1.4. P(X ≤ 0.5)?

P(X ≤ 0.5) = 1 − e^(−0.5/1.4) = 1 − e^(−0.357)

= 1 − 0.700 = 0.300
